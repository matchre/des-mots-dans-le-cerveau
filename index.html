Des Mots dans le cerveau

Niveau

Présentation

Lorsqu'un mot mystère vous est montré, cela active les aires visuelles de votre cerveau. Le but de ce jeu est de reconstituer un mot mystère à partir d'un modèle qui repésente l'activité cérébrale des aires visuelles du cerveau lorsqu'une personne observe ce mot. Une fois le jeu lancé, il vous suffira de survoler avec la souris les différentes zones du jeu pour obtenir des informations complémentaires les concernants. Pour commencer, choisissez un niveau et cliquez sur DEMARRER.

Nombre de coups restants.

Les aires visuelles du cerveau qui sont sollicitées lorsqu'un objet est placé dans le champ visuel d'une personne sont représentées en vert sur ces images.

Cliquez sur les barres de la grille pour reconstituer le mot mystère en un minimum de coup. Lorsque vous cliquez sur le bouton d'aide (niveau facile uniquement), la barre qui s'affiche en rouge vous indique où se trouve l'une de vos erreurs.

Partie supérieure : regardez attentivement ce modèle, il s'agit d'une représentation fidèle de l'activité des aires visuelles cérébrales d'une personne visualisant le mot mystère qu'il vous faut découvrir. Partie inférieure : chaque clic sur la grille de l'écran à gauche de celui-ci allume une cible qu'il vous faudra comparer au modèle.
